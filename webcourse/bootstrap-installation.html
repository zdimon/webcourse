Bootstrap.

Bootstrap is the most popular HTML, CSS, and JS framework for developing projects on the web.

Installation.

npm install bootstrap

Package.json.

{
  "name": "",
  "version": "0.0.1",
  "author": "",
  "description": "blabla",
  "license": "FREEEE",
  "dependencies": {}
}

Installing with saving dependencies.

npm install bootstrap --save

Make sure that you have npm and nodejs installed. If no, you can install it by this commands:

sudo apt-get install nodejs
sudo apt-get install npm

After installing bootstrap you will have it inside node_modules folder. Also, you will have a tool to build css and js files using GRUNT.

GRUNT

Grund is Javascript task runner. It collect your css and js files, compress them and make compact (min) version of them. This process is called as build. To build bootstrap files you have to be inside bootstrap folder and run this command:

grunt build

If you see somethink like

No command 'grunt' found

you need to install dependecsies

npm install

It will get a list of the all dependencies from the package.json file download it and install into node_modules folder.

Available grunt commands

grunt dist

Regenerates the /dist/ directory with compiled and minified CSS and JavaScript files.

grunt watch

Watches the Less source files and automatically recompiles them to CSS whenever you save a change.
